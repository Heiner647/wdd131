Chore Wars Site Plan

Sean Heiner

WDD 131- Final

Overview

Purpose

The purpose of my site is to motivate users to complete everyday chores by turning them into a fun, gamified experience. It allows users to earn points, track progress, and unlock rewards as they complete tasks.

Audience

The intended audience includes teens, young adults, and busy households.

Dynamic elements

It handles features such as assining chores, adding chores, removing chores, and tracking points. JavaScript also allows the interface to change without reloading the page.
